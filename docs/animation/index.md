# Animation

S&box uses Valve's Animgraph system for character animation, providing a powerful node-based animation state machine. The system supports skeletal animation, IK, animated ragdolls, morph targets, and animation events.

Key features:

- **Animgraph** — node-based animation state machines
- **Skeletal animation** and blending
- **Inverse Kinematics** (IK)
- **Animated ragdolls**
- **Morph targets**
- **Animation events**

## Movie Maker

[Movie Maker](/animation/movie-maker/) is a timeline-based editor for animating properties in a scene. It's ideal for in-game cutscenes, editing trailers, coordinating timed events, and creating skeletal animations — all without leaving the editor.
