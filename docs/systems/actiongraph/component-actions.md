# Component Actions

The easiest way to add an *ActionGraph* to your scene is through the Actions Invoker component. Just click Add Component on an object, then you can find it under Actions.


 ![Available actions in an Actions Invoker component.](./images/e3e3bb16-a62f-4d26-9404-1ee144bd90d0.png " =673x259")You can also find action properties in a few other built-in components, like *Colliders* when *Is Trigger* is checked.


 ![Trigger actions in a Collider.](./images/6a9e5a53-c0e8-4cff-8579-58bfad07697d.png)

Each of these start out with an "Empty Action". Simply click on any of those to create a new ActionGraph and start editing it!


 ![First thing you'll see when opening a new action graph.](./images/971daef3-6e48-4bd3-9708-b1927ceedbd9.png " =x614")
